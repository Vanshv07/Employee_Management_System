# Employee Management System (Python + MySQL):-

A simple Employee Management System developed using Python and MySQL.
This project demonstrates how Python can interact with a MySQL database to perform basic database operations such as creating a database, creating tables, inserting records, and retrieving employee data.

It is a beginner-friendly project useful for understanding database connectivity in Python using the PyMySQL library.

# Features:-


✔ Create MySQL database using Python


✔ Create Employee table

✔ Insert employee records

✔ Display all employee records

✔ Display employee record by ID

✔ Use MySQL cursor operations

✔ Exception handling for database errors
